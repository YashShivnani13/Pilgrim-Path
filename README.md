# 🚶 PilgrimPath

### Smart Mobility & Safety Platform for Mahakumbh Pilgrims

PilgrimPath is a transportation and mobility management platform designed to improve the safety, navigation, and overall experience of pilgrims during large-scale events such as Mahakumbh.

---

## 🌟 Overview

Large gatherings like Mahakumbh face significant challenges related to transportation, crowd movement, parking, emergency response, and family safety.

PilgrimPath provides a unified platform that helps pilgrims:

- Navigate safely through crowded areas
- Find optimal routes based on crowd conditions
- Locate available parking spaces
- Stay connected with family members
- Access emergency services instantly
- Report and track lost & found cases

---

## ✨ Features

### 🗺️ Smart Navigation
- Crowd-aware route recommendations
- Alternative route suggestions
- Real-time mobility guidance

### 📍 Live Crowd Map
- Visual crowd density indicators
- Sector-wise crowd monitoring
- Important location markers

### 👨‍👩‍👧 Family Tracker
- Create family groups
- Join existing groups
- Track family members
- Coordinate meeting points

### 🚨 Emergency SOS
- One-tap emergency assistance
- Quick access to emergency contacts
- Nearby medical and police support

### 🔎 Lost & Found
- Report missing persons
- Report found persons
- Report missing items
- Browse reports

### 🅿️ Smart Parking
- Parking occupancy monitoring
- Recommended parking zones
- Parking navigation assistance
